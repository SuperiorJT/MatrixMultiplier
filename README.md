## MatrixMultiplier
This program will multiply two matrices together to obtain a dot product.

<ol>
  <li>_Parsing_ data from a file</li>
  <li>Calculating dot product of two matrices</li>
  <li>Creating **Threads** and _joining_ them </li>
</ol>
